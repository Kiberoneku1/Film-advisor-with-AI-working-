# Film-advisor-with-AI-working-
## Описание проекта
The site recommends movies based on the mood you enter into the input field.
## Стек
I used HTML, CSS, and Python to develop this service. The libraries employed include Flask, transformers, pipeline, and hf_xet. 
The neural network model for movie ideas is from sberbank-ai, while the model for sentiment analysis - blanchefort.
## installing libraries
To install the libraries, you need to enter this command:
```
pip install -r requirements.txt
```
Or install the library separately:
```
   pip install Flask
   pip install transformers
   pip install pipeline
   pip install torch
 ```
